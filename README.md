## 处理路由
现在的页面由于多种原因，都在做spa，因此router 必不可少。 
项目中使用的react-router + react-router-redux: 前者是业界标准，后者可以同步 route 信息到 state，
这样你可以在 view 根据 route 信息调整展现，以及通过 action 来修改 route 。
